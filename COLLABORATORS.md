# Collaborating with VibeTrack

VibeTrack is an open-source project. We welcome contributions from developers worldwide.

## Ways to Collaborate

### Code Contributions
- Fix bugs
- Add features
- Improve documentation
- Write tests

### Non-Code Contributions
- Report issues
- Suggest features
- Write tutorials
- Create themes/translations
- Promote the project

## Getting Started

1. Join our community discussions
2. Pick an issue labeled `good first issue`
3. Ask questions if stuck
4. Submit your first PR

## Recognition

All contributors are recognized:
- GitHub Contributors page
- Release notes
- Special mentions for significant contributions

## Roles

### Contributors
Anyone who contributes code, docs, or feedback.

### Maintainers
Project leads who:
- Review and merge PRs
- Manage releases
- Make strategic decisions
- Ensure project health

## Communication

- GitHub Issues: Feature requests and bug reports
- GitHub Discussions: Q&A and general discussion
- Email: contact@vibetrack.app
